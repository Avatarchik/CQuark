# CQuark
西瓜（CQuark）是一个简单的C#语法的脚本语言。前身是曾经自己写的一个脚本语言以及C#LightEvil（https://github.com/lightszero/cslightcore）。
西瓜语法与C#完全一致，可以用于Unity热更新。

如果在非Unity环境下运行，直接删除Asset下_Unity文件夹即可使用
